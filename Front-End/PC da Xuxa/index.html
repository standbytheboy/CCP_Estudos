Exercício de JavaScript

É uma linguagem de programação fracamente tipada utilizada para fazer a interatividade em páginas da Web. É uma linguagem scripting, ou seja, é interpretada e não depende de compilação para funcionar, é lida diretamente pelo navegador sem precisar de conversões ou traduções.

Foi criado na década de 1990 pela Netscape, com o objetivo de atender a necessidade de sites mais dinâmicos,que até então eram sem muita interatividade, exceto pelos hiperlinks. Recebeu o nome “JavaScript” em 1995, quando o projeto foi comprado pela Sun Microsystems (fundadora do Java).
