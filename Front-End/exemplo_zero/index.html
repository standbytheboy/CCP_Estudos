Texto de exemplo escrito com a tag "p" para formatar com CSS. Nesse texto aplicamos a fonte que baixamos para o nosso projeto, a propriedade "border", alteramos a cor do texto e a de fundo.

Texto com atributo "id" ao invés de "class", o que nos permite colocar outra formatação. Vale lembrar que um "id" não se repete.

Esse texto funciona para entender como funciona a herança no HTML e CSS. A herança é a capacidade de uma propriedade ser herdada de outra tag.
